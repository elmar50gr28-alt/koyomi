# Western Astrology Specification

## Required Person Data

- Birth date
- Birth time
- Birthplace
- Latitude and longitude
- Time zone at birth

## Conditional Data

- Birth time unknown
- Time accuracy
- House system
- Daylight saving time
- Location accuracy

## Readiness Notes

If birth time is unknown, natal planets can be used but houses and angles must be marked low confidence or unavailable.

## Output Requirements

- Show house system.
- Show time and location basis.
- Mark approximate or unknown time clearly.
