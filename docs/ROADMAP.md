# KOYOMI Roadmap

## 1. 目的

このロードマップは、KOYOMIを安全に拡張し、占術・UI・保存・総合鑑定・ウェアラブル対応まで段階的に完成させるための開発順序を定める。

原則として、基盤を固めてから機能を増やす。

---

## 2. 現在地

現在のKOYOMIには、以下が存在する。

- ホーム画面
- 複雑暦時計UI
- 四柱推命
- 奇門遁甲
- 相性鑑定
- 運勢年表
- 世界の暦
- 人物台帳
- タロット
- ルーン
- 総合鑑定
- みつのめ姉さん人格
- PWA
- Service Worker
- 四柱推命モジュール
- 四柱推命データベース
- 自動テスト
- GitHub Actions

ただし、`app.html`への機能集中が大きく、今後の開発では段階的な分割が必要である。

---

## 3. 開発フェーズ

## Phase 0：基盤安定化

目的：

- 壊れにくい開発環境を作る
- 変更時に異常を検知できるようにする
- GitHub運用を定着させる

実施項目：

- Service Worker改善
- 静的検証テスト更新
- モバイル回帰テスト
- 四柱推命テスト
- Pages公開前チェック
- アーキテクチャ文書作成
- ロードマップ作成

状態：

- Service Worker改善：完了
- 自動テスト更新：完了
- アーキテクチャ文書：完了
- ロードマップ：進行中

- 干合

- 冲
