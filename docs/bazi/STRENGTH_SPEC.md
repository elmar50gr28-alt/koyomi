# Bazi Phase 2 Strength Spec

Phase 2 keeps strength evaluation dimensional. It does not decide day-master strength from one score.

Implemented structured factors:

- month command and seasonal phase
- roots by hidden stem role and pillar position
- exposed stems
- element distribution
- climate state
- qi flow
- day master strength level

Returned levels:

- `extremely-weak`
- `weak`
- `slightly-weak`
- `balanced`
- `slightly-strong`
- `strong`
- `extremely-strong`
- `indeterminate`

Each result includes supporting factors, opposing factors, unresolved factors, school results, confidence, and evidence IDs.
