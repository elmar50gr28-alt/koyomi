# KOYOMI Bazi Source Policy

KOYOMI must be explainable. Every rule should link to source IDs or internal evidence IDs.

Phase 1 stores bibliographic and concept-level seed entries only. It does not scrape or copy unauthorized modern web articles.

Allowed source data:

- bibliographic metadata
- public-domain classical references
- short, reviewable evidence units where legally safe
- KOYOMI-created rule summaries clearly marked as rules, not sources

Disallowed:

- unauthorized modern translations
- long copied passages
- AI-generated text labeled as a classical source
- rules without review status
