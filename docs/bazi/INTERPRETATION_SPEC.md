# Bazi Phase 3 Interpretation Spec

Phase 3 adds structured interpretation data. It does not create deterministic fortune statements.

Implemented categories:

- career
- finance
- relationship
- family
- health

Each category returns:

- tendency
- conditions
- confidence
- source IDs
- review status

Luck interpretation is also structured by scope:

- decade luck
- annual luck
- monthly luck

Annual and monthly luck are schema-ready and marked for review where precise calculation is not yet implemented.
