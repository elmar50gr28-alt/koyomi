# Bazi Phase 2 Luck Cycle Spec

Luck cycles use the Phase 1 chart and month pillar as the deterministic base.

Implemented:

- forward/reverse direction candidate
- direction rule ID
- approximate start age
- approximate start date
- eight ten-year cycles
- cycle stem and branch
- warnings for approximate solar-term start timing

Exact luck start dates require astronomical solar-term entry times and remain a later phase item.
