# Bazi Phase 2 Yongshen Spec

Yongshen methods are separated by method.

Implemented methods:

- `support-control`
- `climate`
- `mediation`
- `illness-medicine`
- `pattern`
- `assistant-god`

The integrated result compares method outputs and reports conflicts. It does not collapse different schools or methods into a single unsupported final answer.
