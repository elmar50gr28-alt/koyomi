# KOYOMI Bazi School Differences

Phase 1 registers these school profiles:

- `traditional-ziping`
- `yuanhai-ziping`
- `sanming-tonghui`
- `ziping-zhenquan`
- `ditian-sui`
- `qiongtong-baojian`
- `taiwan-modern`
- `hongkong-modern`
- `japan-taizan`
- `modern-score`
- `koyomi-integrated`

Each profile can define:

- day boundary
- zi-hour handling
- strength method
- luck start method

The engine keeps school results comparable. It does not silently choose one modern scoring method as the only truth.
