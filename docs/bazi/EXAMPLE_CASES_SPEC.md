# Bazi Phase 3 Example Cases Spec

Example cases are stored in `data/bazi/example-cases.json`.

Phase 3 uses synthetic regression cases only. No real personal data is included.

Each case includes:

- `caseId`
- input profile
- adopted school IDs
- expected structured result flags
- evidence IDs
- review status

Cases are for regression and evidence wiring, not proof of predictive accuracy.
