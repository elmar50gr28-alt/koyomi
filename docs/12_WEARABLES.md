# Wearables Specification

## Apple Watch Assumption

Apple Watch support is a future requirement. The watch should not receive full readings; it should receive short, actionable signals.

## Garmin Assumption

Garmin support is future-facing and should be lightweight, low-power, and offline friendly.

## Wearable Content

- Today's signal
- One recommended action
- Warning time band
- Short reminder
- Link back to evidence on phone

## Data Rule

Wearables must not become the primary storage location for sensitive personal data.
