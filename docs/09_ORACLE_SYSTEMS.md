# Oracle Systems Specification

## Included Systems

- Tarot
- Runes
- Mesopotamian reconstruction/oracle layer
- Sound/name symbolic systems
- Other symbolic oracle layers

## Required Data

- Question or theme
- Spread/method
- Optional target `personId`

Birth information is not required for Tarot or Runes.

## Compatibility Readings

Compatibility readings use:

- Two shared profiles
- Relationship type
- Consultation purpose
- Missing information according to selected systems

## Evidence Layers

Clearly separate:

- Historical source
- Reconstruction/calculation
- Modern symbolic creation
- AI explanation
