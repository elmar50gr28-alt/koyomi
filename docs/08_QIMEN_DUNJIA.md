# Qimen Dunjia Specification

## Required for Main Reading

- Divination date/time
- Reference location
- Consultation purpose

## Conditional Data

- Current heading
- Device compass
- Position of self and other party
- School setting

## Relationship to Person Profiles

Birth information may be used for combined readings with systems such as Shichusuimei, but it is not required for standalone Qimen.

## Safety

Do not request location or compass permission on startup. Ask only after explicit user action.
