# Bazi derived information core extraction

Hidden stems, ten gods, twelve stages, seasonal classification, empty void,
element balance and per-pillar enrichment are grouped in
`src/bazi/chart/derived-info.js`.

The chart builder calls this core after the four foundation pillars are
calculated. The original operation order, default school identifier, weights,
rounding and result shape are preserved.

Strength, pattern and favorable-element rules are outside this extraction and
remain unchanged. UI, profile ledger, persistence, encryption, backup and the
service worker are also unchanged.
