# Name Divination Specification

## Required Person Data

- Name
- Family/given name structure
- Reading
- Characters used
- Adopted glyph form

## Conditional Data

- Old glyphs
- Variant glyphs
- Original spelling
- Romanization
- Manual stroke overrides
- School-specific stroke method

## UI Rule

Name divination details belong in detailed settings unless the user explicitly chooses a name-reading workflow.

## Preservation

Hidden name-divination data must be preserved even when not shown on the normal profile screen.

## Numerology

Required:

- Birth date
- Birth name or active/use name

## Gematria / Kabbalah

Required:

- Original spelling
- Writing system
- Transliteration method

Conditional:

- Birth date
