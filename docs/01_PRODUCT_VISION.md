# KOYOMI Product Vision

KOYOMI aims to become a world-ready jyutsusu operating system for daily life.

## Product Promise

KOYOMI helps the user read today, choose today, and live today. It converts complex traditional systems into actionable daily guidance while keeping evidence visible.

## Users

- Beginners who need simple inputs and plain-language guidance.
- Practitioners who need schools, roots, precision, and auditability.
- Global users with non-Japanese names, overseas birthplaces, and diverse time systems.
- Future wearable users who need short signals, reminders, and quick confirmation.

## Product Shape

- Mobile app first.
- Offline-capable PWA.
- One shared person profile.
- Multiple deterministic divination engines.
- Explainable result layer.
- Mitsunome as narrator/translator, not calculator.

## Success Criteria

- A first-time iPhone user can register a person and start a reading quickly.
- A professional user can inspect why the result was produced.
- Saved data survives app updates.
- No calculation result depends on hidden AI improvisation.
