# KOYOMI v300 Foundation Edition

## 起動
`index.html` をHTTPSサーバーで公開して開きます。iPhoneではSafariの「ホーム画面に追加」を利用できます。

## 構成
- `index.html`: 軽い入口と今日の行動
- `today.html`: 常時動く暦機構UIプロトタイプ
- `app.html`: 既存の詳細鑑定、奇門遁甲、相性、年表、世界暦、人物台帳
- `service-worker.js`: 基本ファイルのオフラインキャッシュ

## 重要
Foundation版は既存資産を失わず、今後の検証と強化を行う基盤です。全占術の科学的的中率や、全流派の完全再現を保証するものではありません。医療・法律・金融・安全に関する判断は専門家へ相談してください。
